# 斗铠

> **警告：以下内容属于*CrCraft Downloadable Content*，可能会破坏您的纯生存体验，请谨慎阅读。**
>
> *注意：制作斗铠后，即默认至少已认真阅读本文所有内容，并且保证严格遵守规定。*

**斗铠**是一种基于原版**Minecraft**制作的强力装备，其灵感来源于《斗罗大陆III：龙王传说》。

## 关于斗铠

斗铠在制作时需要给其命名，一字斗铠为一个字，每升级一次斗铠就加一个字。斗铠名字的颜色可以选择和盔甲纹饰相近的颜色，字样可以设置斜体、粗体、下划线、删除线。斗铠的信息框会显示制作者署名、斗铠级别和版本，且可以自定义内容，署名和自定义内容的字样和颜色可任选。一、二、三、四字斗铠的斗铠版本分别用白、黄、紫、红色（参考了《斗罗大陆》的魂环颜色）的文字表示。

**严禁用任何手段擅自给斗铠增加额外附魔或去除原有魔咒**。斗铠不兼容 *经验修补* 魔咒，如果原装备有，则会被去除。只有下界合金质斗铠可以提供 *盔甲值* 和 *击退抗性* 的增幅，钻石斗铠升级至下界合金斗铠后也能提供，但每次升级一件斗铠盔甲最多增加 `0.25` *盔甲值* 和 *击退抗性*。

每个玩家最多拥有一套斗铠（全套盔甲和一把武器），如果需要立即制作新的斗铠，则必须销毁原本斗铠。

*命令武器*（见[凋灵风暴](https://modrinth.com/mod/crackers-wither-storm-mod)）可以用于制作斗铠武器，拥有更高的基础伤害，但无法升级至下界合金质，其 *不可破坏* 属性会被更改为额外增加3级 *耐久* 魔咒，最高不超过 `耐久 X`。

斗铠制作最多需要一周左右的时间。

一般情况下，不提倡使用斗铠**影响、干扰、甚至攻击**没有斗铠的玩家（熊孩子、外挂除外）。

## 斗铠所需经验及材料

全套斗铠的升级材料。（包括头盔、胸甲、护腿、靴子和武器，且都需要提供前一级的装备用于升级，装备需要拥有诅咒类魔咒。）

升级斗铠时原装备要拥有至少75%耐久度。

| 消耗 | 一字斗铠                      | 二字斗铠                                   | 三字斗铠                                                                               | 四字斗铠                                                 |
| ---- | ----------------------------- | ------------------------------------------ | -------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| 经验 | 50级                          | 100级                                      | 150级                                                                                  | 200级                                                    |
| 材料 | 纹饰锻造模板\*4+铸锭或晶体\*4 | 对应颜色潜影盒*40 或 末影箱\*256（二选一） | 龙息\*192+鹦鹉螺壳\*4 或三种蛙明灯各4个 或 猪灵头颅\*8 或 滞留型神龟药水\*12（四选一） | 龙蛋\*1 或 龙首\*4+信标\*8+鞘翅*4（满4个魔咒）（二选一） |

## 修复斗铠

玩家擅自在铁砧上修复斗铠会显示“*过于昂贵！*”，但是可以通过向服主提供修复所需原材料并消耗经验的方式间接使用铁砧修复（每个下界合金斗铠只需一个下界合金锭就可修满耐久）。

间接使用铁砧修复斗铠时，不会因为所需经验超过39级而修复失败。

斗铠使用铁砧修复时的初始累计惩罚取决于斗铠级别。

|              | 一字斗铠 | 二字斗铠 | 三字斗铠 | 四字斗铠 |
| ------------ | -------- | -------- | -------- | -------- |
| 初始累计惩罚 | 15       | 7        | 3        | 1        |
